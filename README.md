# Blitzortung Lightning Cluster
A Web Application that shows the heat maps of lightning strike hits using BFS, Greedy, and Prims algorithms

## Installation / Run
1. Download the ZIP
2. Pip install the requirements.txt
3. Run the app.py and wait for the Chromium window with the Blitzortung website to open and start providing websocket messages
4. To reach to the Web Application go to http://127.0.0.1:8080 or go to the designated link provided by flask in the console when running
5. Refresh the page to find the new lightning strikes and their heat clusters on the map
